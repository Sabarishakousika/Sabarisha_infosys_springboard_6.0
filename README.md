# Infosys Springboard 6.0 – Intelligent Wealth Management Platform

## 🌟 Project Vision

The **Infosys Springboard 6.0 Wealth Management Platform** is a comprehensive full-stack financial management system designed to help users track, analyze, and optimize their financial portfolios.

This project reflects my strong interest in **financial technology (FinTech), data-driven decision making, and modern web application development**. It integrates powerful backend services with an intuitive and responsive frontend interface to deliver a seamless user experience.

Through this project, I explored how technology can empower individuals to better understand their financial health, plan future investments, and make informed financial decisions.

---

## 🎯 Project Objectives

The goal of this project is to build a scalable platform that enables users to:

* Track their investments and financial assets
* Monitor portfolio performance
* Manage financial goals
* Analyze financial trends through dashboards
* Generate financial reports
* Simulate investment scenarios
* Receive actionable insights from their financial data

The platform demonstrates how modern full-stack technologies can be combined to build a **secure, scalable, and user-centric financial application**.

---

## 🧠 Key Features

### User Management & Authentication

Secure user authentication using **JWT-based authentication** ensuring protected access to financial data.

### Portfolio Tracking

Users can track their investments, holdings, and asset allocations through a unified dashboard.

### Financial Goal Management

The system allows users to create, monitor, and update long-term financial goals.

### Analytics Dashboard

Interactive visualizations provide insights into portfolio performance, asset allocation, and market trends.

### Investment Simulations

Users can simulate different investment scenarios and evaluate potential outcomes.

### Notifications & Reporting

Automated reporting tools help users generate portfolio reports and receive notifications about important updates.

### Admin Dashboard

Administrative tools allow management of users, system activity monitoring, and performance insights.

## 🏗️ System Architecture

The project follows a **modern full-stack architecture**, separating responsibilities between frontend and backend services.

**Frontend Layer**

* Built with **React + TypeScript**
* Styled with **Tailwind CSS and ShadCN UI**
* Uses reusable components and modular design
* Provides responsive user interfaces and interactive dashboards

**Backend Layer**

* Developed with **FastAPI**
* RESTful API architecture
* Uses **SQLAlchemy ORM** for database interaction
* Secure authentication using **JWT tokens**
* Background task support using **Celery**

**Data Layer**

* Supports **PostgreSQL / SQLite**
* Handles financial records, user profiles, and analytics data

This architecture ensures the system is **modular, scalable, and maintainable**.

## ⚙️ Installation & Setup

### Clone the Repository

```
git clone https://github.com/Sabarishakousika/infosys_springboard_6.0.git
cd infosys_springboard_6.0
```

### Backend Setup

```
cd fastapi_backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup

```
cd wealth_frontend
npm install
npm run dev
```

After starting both servers, the application will be accessible locally.
